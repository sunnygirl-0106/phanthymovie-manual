总体流程概览

PhanthyMovie 是一套面向 AI 短剧工业化量产的 SaaS 平台。它把"剧本 → 成片"的全流程拆成一条可被流水线管理的工序，让一个人也能在 1 天内完成原本需要一个剧组才能完成的工作。

上传剧本，剩下的交给流水线。

整套产品的使用顺序是规则化的——先把图片打磨到位（剧本建库 → 素材草图 → 融合草图 → 批量出图），再进入视频环节（拍摄台 / 视频工坊）。图片质量决定视频质量，因为所有视频都是基于关键帧或首尾帧生成出来的。

关键提醒
图片质量非常重要，因为所有图都是视频的"关键帧"或"首尾帧"。任何在视频阶段出现的镜头不一致、人物变形、风格漂移，几乎都能追溯到前面四步图片没做好。先把图片磨到 90 分，再去做视频。
第一部分

登录与落地页

打开浏览器访问 cloud.phanthy.com/phanmovie 即可进入落地页。落地页用电影海报式的大画面承载主标语 "你的专业制片厂，所想，即所得"，下方陈列产品的四大典型用例。

必读
登录是使用 PhanthyMovie 的第一步，也是后续所有功能的前置条件。没有登录，所有的项目、资产、计费数据都无法关联到你的账户，平台会拒绝任何生成请求。

点击右上角"注册/登录"后，会弹出一个居中的登录卡片，提供两种登录方式。

  1. 1
    在落地页右上角点击"注册/登录"。
  2. 2
    在弹出的卡片里选择登录方式（默认是"用户密码"）。
  3. 3
    填写凭证 → 点击蓝色"登录"按钮。
  4. 4
    登录成功后会进入新版首页（创作中心）。
注意事项
  • 用户名 / 密码由管理员在"用户管理"里下发，首次登录后建议立刻去设置里改密码。
  • 手机验证码模式需要先在管理员处把手机号绑到账号上。
  • 如果一直停在登录卡片，说明账号未授权或后端未连上，去找管理员复核后端状态是否绿灯。
新版首页

创作中心与项目管理

登录成功后，你会进入 PhanthyMovie 的全新首页。顶部有四个 Tab：创作中心、项目管理、公共资产库、素材广场。下面重点介绍最常用的两个——创作中心（选择一种方式开始创作）与项目管理（管理你名下的所有项目）。页面右上角固定显示充值中心、星钻余额、通知、头像（个人中心）。

创作中心

「创作中心」是登录后的默认落地页，主标题 "开启您的创作旅程"。中间并排陈列三张创作方式卡：

①
AI 智能片场

即原来的"工作流"，已有剧本 / 小说时的首选：一键转分镜画面，自动识别角色与场景，产出可直接拍摄的全套镜头。点"开始 →"进入图片四步法。

②
全自动 AI 生成

输入主题、风格、片长，AI 自动生成剧本、分镜、画面与声音，一键产出完整短片。当前即将开放。

③
无限画布

即原来的"画布"，在自由画布上随手拼接素材、参考图与 AI 节点，边探索边创作，适合非线性表达。点"开始 →"进入画布。

说明
无论从「AI 智能片场」还是「无限画布」点"开始"，都会先弹出"创建新项目"窗口——填写项目名称、选择画面比例（16:9 横屏 / 9:16 竖屏）与风格（写实 / 电影质感 / 3d / 动漫…），点"创建项目"即进入对应模式。

平台作品

三张创作方式卡下方是新版新增的 "平台作品" 区，用来"看看其他创作者最近发布的短片"，右上角可在 最新 / 热门 之间切换。每部作品以视频卡片陈列，显示封面、时长、标题、作者与点赞数；鼠标悬停即可预览，点爱心就能为喜欢的短片点赞。页面底部还有一条提示"所有生成的项目将自动保存至您的个人库中"。

项目管理

点顶部 "项目管理" 进入 "我的项目" 页，以网格列出当前账号下的所有项目，每张卡片显示封面、名称、创建日期，右下角 ⋮ 可重命名 / 删除。右上角提供关键词搜索、批量操作、时间倒序，以及"+ 创建新项目"。

点进任意项目会弹出 "创作模式" 窗口（"同一项目下共享项目资产数据"），三种模式可选：

①
工作流

适合流程化拆解与自动化生成，强调创作链路与环节衔接。点"立即进入"进入图片四步法。

②
全自动 AI 生成

适合自动拆图、批量出图与生成，偏高效率批处理。当前即将开放。

③
无限画布

进入画布列表，用节点式创作方式组织镜头、图片与全景生成。点"立即进入"进入画布。

要点
弹窗底部提示"当前仅开放无限画布模式，后续模式会直接复用当前项目资产数据"——同一项目里无论切换哪种模式，素材与资产都是打通共享的。
第二部分

工作流 · 图片四步法（主控制台）

主控制台是 PhanthyMovie 的"生产驾驶舱"，也就是「工作流」模式的工作区。从「创作中心 → AI 智能片场」，或在「项目管理」中打开项目并选择"工作流"，都会进入这里。整个产品的核心使用顺序全部围绕它展开——上面有 4 个大卡片，对应图片生成的 4 个步骤，必须严格按 1 → 2 → 3 → 4 的顺序走完，才会进入视频环节。

①
左侧导航栏

主控制台、画布、我的项目、资产库、剧本拆解、拍摄台、视频工坊、试验台、小工具。

②
底部管理员入口

用户管理、账户管理、计费统计、团队管理、系统设置、运营数据看板（普通用户看不到）。

③
主区域·四步流水线

第一步剧本构建 → 第二步素材草图 → 第三步融合草图 → 第四步批量出图。

注意
四步严格顺序依赖：前一步未完成，后一步是灰的、点不动。后端状态显示"未连接"时任何任务都不会推进，先排查连通性再操作。同一项目可多次重跑某一步，但建议每次只跑一步、便于排错。
第二部分 · 第一步

剧本构建

剧本构建是整条流水线的起点。你只需要把剧本文本喂给系统，AI 会自动完成两件事：①结构化拆解（把整本剧本拆成 集 → 场 → 镜头 三级结构）；②资产识别（从剧本里抽出 角色 / 服装 / 场景 / 道具 清单）。

注意
这一步是纯文字，不会生成任何图片，你看到的所有输出都是文字结构——也因此不烧图片 / 视频额度，可以放心反复跑。
  1. 1
    进入主控制台 → 点击"第一步｜剧本构建"卡片。
  2. 2
    在弹窗里选择剧本来源：上传文件（.txt / .pdf / .docx）或粘贴文本。
  3. 3
    系统实时显示字数统计，可点"AI 预估集/场"预估消耗的星钻。
  4. 4
    检查无误后点击右下角"开始自动建库"。

提交后跳回主控制台，日志面板逐行打印解析进度——依次跑「剧本拆解 → 镜头生成 → 项目梗概 → 素材补全 → 融合素材」等子任务，每条带时间戳和模型名，通常 1~3 分钟完成。

建库完成后，到左侧导航的"剧本拆解"里可以进一步精修（顶部多个 Tab，逐集逐场核对拆解结果），精修完成后回到主控制台进入第二步。

使用建议
  • 第一遍请上传完整的整本剧本解析，不要只传片段或单集，否则拆解结果不完整。
  • AI 识别出"路人甲 / 黑衣人"这种含糊命名，建议到资产库手动改名，避免后续融合素材爆炸。
  • 原文里若已有 [人物介绍]、[场景] 等结构化标签，AI 会拆得更准。
注意事项
  • 删除一集会级联删除该集下所有场、镜头和素材，操作前请确认。
  • 重新解析会覆盖原有结构化数据，但已经生成的图片不会丢。
第二部分 · 第二步

素材草图

素材草图是给"建库时识别出的每一个素材"出一张参考图，让你在花大钱出关键帧之前，先用便宜的草图验证："AI 理解的这个角色 / 这件衣服 / 这个场景，是不是我想要的？"

要点
它解决的核心问题是：AI 描述对不对。如果描述错了，后续所有图都会跟着错。
  1. 1
    在主控制台点击"第二步｜素材草图"卡片，弹出费用预估窗口。
  2. 2
    配置模型（如 phan nano Image 3）、清晰度（1K/2K/4K）、画面比例（角色三视图常用 1:1，竖屏短剧用 9:16）。
  3. 3
    看清"AI 草图（基础素材）"的 次数 / 原价 / 折后 价格。
  4. 4
    确认无误后点击"确认并启动"，约 1~3 分钟完成。

完成后到 资产库 / 剧本拆解 → AI 草图 Tab 里查看每个素材的参考图。

使用建议
  • 第一次出图建议 1K + 一次出多张，挑出基本对的，再用 2K/4K 复跑最终图。
  • 某素材明显画错（性别错、衣服颜色不对），先回资产库修描述，再单素材重跑。
  • 草图阶段的"挑图 + 重跑"是整个项目最具性价比的环节，多花点时间在这里，后面省一大块。
注意事项
  • 本步骤产出的是每个素材一张图，不会自动挂载到任何镜头。
  • 改完素材描述，已生成的草图不会自动重出，需进入资产库点"重新生成"。
第二部分 · 第三步

融合草图

融合草图解决一个非常实际的问题：单素材好看，组合到一起就不一定好看。比如"苏可"（角色）穿"周末居家卫衣"（服装）拿"手机"（道具），三个素材分别都出得好，但组合出来可能比例不对、风格冲突。融合草图就是把这些组合提前打样，让你在批量出图之前验证搭配效果。

  1. 1
    完成第二步并对素材草图满意后，点击"第三步｜融合草图"卡片。
  2. 2
    在弹窗配置模型、清晰度（同样建议 1K）、画面比例。
  3. 3
    看清"AI 草图（组合素材）"的预估次数和费用。
  4. 4
    点击"确认并启动"，日志面板会逐条打印"融合素材 / 角色+服装 / 着装角色+道具"等进度。
典型场景
  • 同一角色在不同场换不同衣服 → 系统自动生成多个"角色-服装"融合素材，需一一确认。
  • 角色佩戴独特道具（手机、酒杯、武器）→ 生成"着装角色+道具"融合素材。
使用建议
  • 融合素材数量通常是基础素材的 1~3 倍，可能比第二步多得多，做好心理准备。
  • 某个融合明显错了（如杯子飘在空中），别硬重跑，先回资产库把基础素材修好再重新触发融合。
注意
同样是先调参数再确认启动，否则会被自动按默认配置扣星钻。融合素材是"两两组合"，费用比第二步显著更高，事前看一眼预估金额。
第二部分 · 第四步

批量出图

前三步铺垫的图都属于"参考图"，真正用于视频的关键帧 / 首尾帧在第四步才会出。这是整条流水线最重、也最贵的一步，系统会自动做 5 件事：

1自动配素材把项目/集/场层级资产分配到每个 Shot
2生成视频提示词LLM 为每个场景生成视频 prompt
3关键帧生图图像 AI 生成每个镜头的关键帧
4生成首尾帧提示词LLM 为每个镜头生成首/尾帧提示词
5首尾帧生图图像 AI 根据提示词生成首尾帧图片

完成后，所有镜头的关键帧和首尾帧都备好了，可以直接进入拍摄台 / 视频工坊出视频。

  1. 1
    点击"第四步｜批量出图"卡片，弹窗列出 5 个阶段的可调参数与预估。
  2. 2
    弹窗实时显示每阶段"次数 / 原价 / 折后"和最终合计金额。
  3. 3
    检查总价没问题后，点击"确认并启动"。
  4. 4
    日志面板持续刷新各子任务状态，全部打勾即代表全部图片素材已就绪。
使用建议
  • 第四步是整条流水线最贵的一步，启动前一定看清总价。
  • 启动前确认资产库里没有错的角色 / 服装，否则会有一批镜头白跑。
  • 中途某些镜头失败，不必整体重跑，可到拍摄台单独补图。
注意事项
  • 任务被取消或系统重启后会从中断处续跑，不必从零重头。
  • 单个项目同一时刻只能跑一个"批量出图"任务，第二次点会被拦截。
  • 完成后请进入"拍摄台"逐场抽查关键帧 / 首尾帧效果，再决定是否进入视频环节。
第二部分 · 资产库

资产库 · 集中管理与精修

资产库呈现当前项目所有素材的集中视图，顶部按类别分 Tab——角色、服装、场景、道具、着装角色、持道具角色、音频，以及"全部"。每张卡显示缩略图、类别、名称和详细属性；右下角永远有一张 + 新建 占位卡，方便手动补素材。

把鼠标移到任意素材卡上，会出现"编辑 / 重新生成 / 删除"操作。对不满意的素材可直接重新生成，也可以"添加图片 / 粘贴图片"把你自己的图喂进来。

建议
如果对某个自动生成的资产不满意，在这里做微调（改描述重生、上传自己的图、重新编辑）是最快的；改完描述后已生成的图不会自动重出，需要点"重新生成"。
第三部分 · 视频

拍摄台

第四步完成后，所有图片素材都已"挂载"在镜头上，接下来就是把图片变成视频。视频环节主要在拍摄台和视频工坊完成，配套还有试验台做临时试验、小工具辅助维护。

拍摄台是"批量出图"阶段的可视化工作区，也是视频生成的发起入口。它把第四步内部的子任务全部摊开成顶部 7 个 Tab，让你能在任一环节回查 / 修改 / 重跑：

自动配素材视频提示词关键帧生图首尾帧提示词首尾帧模式关键帧视频生成首尾帧视频生成

自动配素材

典型场景
跑完第四步后过来抽查，发现某场少配了道具 → 单场重跑挂载。

视频提示词

建议
进入视频生成前，先用 1~2 分钟通读提示词，把明显跑偏的地方手动改一下，比直接出视频再重做划算得多。

关键帧生图

以网格画廊形式展示当前场所有镜头的关键帧，每张都能点开放大、单图重生。左侧带素材库分类侧栏。

首尾帧提示词 / 首尾帧模式

首尾帧提示词：选中镜头后分"首帧/尾帧"两块文本编辑器，可 AI 生成、批量生成。首尾帧模式：分两栏陈列"首帧候选 / 尾帧候选"，每个候选可"上传图片 / 从素材库选"。对结果敏感的镜头，建议手动微调打光、运镜等关键字，能大幅提升成片观感。

关键帧视频生成 / 首尾帧视频生成

这两个 Tab 是真正把图变成视频的地方，也是短剧成片最常用的入口。顶部参数条可设 比例 9:16 / 分辨率 720p / 时长 5s / 模型 / 配音，主区可挂关键帧、视频参考、音频参考，并带出"视频提示词"文本框。确认后点"生成视频"即可。

注意
视频生成是平台最贵的环节，每提交一条都会扣星钻，提交前盯紧右上角的"本次预估消耗"。

视频预览（成片）

生成完成后即可直接预览。这是一段 9:16 竖屏成片，播放器下方通常带下载 / 添加到当前画布 / 截取当前帧 / 查看参数等功能。到这里，一个"上传剧本 → 自动出图 → 拍摄台出视频"的完整流程就跑通了。

建议
拍摄台是"看素材 + 微调"的地方，不是"重做素材"的地方。
第三部分 · 视频

视频工坊

视频工坊是视频生成任务的提交、监控与管理中心。拍摄台是"按镜头逐个出视频"的精修工作区；视频工坊则是"批量提交 + 全局监控"的指挥中心。也可以绕过前面所有步骤，直接在这里手动生成想要的视频。

左侧任务队列

顶部 全部/完成 Tab，可按备注、集、场、镜号、时间戳筛选，支持一键下载、回收站；任务卡显示状态 + 备注 + 创建时间。

主区生成器

填内容备注 → 设参数（比例/分辨率/时长/模型/配音/联网）→ 在"首尾帧模式"和"参考模式"间切换 → 配好参考后点"加入队列"。

典型场景
  • 单镜头补拍：拍摄台某镜头跑挂了 → 视频工坊手动调参数重新提交。
  • 临时性创作：临时来一个广告 / 片段 → 不需建库，直接配好首尾帧 + 提示词加入队列。
  • 横向比对：同一镜头用不同模型 / 参数各跑一次，全丢进队列，最后挑最好的。
注意
提交前一定确认生成参数（特别是模型和时长），它们直接决定计费；这种手动方式没有素材一致性保护，适合单镜头 demo，不适合整剧批量。任务队列支持批量取消 / 重试，但取消后已消耗的星钻不退。
第三部分 · 视频

试验台 & 小工具

试验台

试验台相当于一个内置的 Playground，可以在任何时间做"试验性"的事：临时测提示词、单图快速出图试风格、写文案、或与 AI 对话 brainstorm。它不写入项目数据，是一个独立的快速创作沙盒，方便你在不污染主流水线的前提下试新东西。

界面为 ChatGPT 式会话体验：左侧"新建会话"+ 历史列表，主区聊天区，底部可切换 生文/生图 模式、选模型、设并发数，支持上传参考图、用 @ 引用资产库素材，Ctrl+Enter 发送。

注意
试验台出的图不会自动入资产库，要手动保存或截图；同样会扣星钻（除非已豁免）。

小工具 · 图片角度旋转

侧栏的"小工具"是一些和主流程关系不大、但用起来很顺手的独立功能。当前主要是图片角度旋转：上传一张图，通过 3D 视角预览拖动手柄或直接选参数，就能生成对应视角的图。

水平角度

0°正面 / 45°右前 / 90°右侧 / 135°右后 / 180°背面 / 225°左后 / 270°左侧 / 315°左前

垂直角度

俯视 −30° ~ 平视 0° ~ 仰视 60°

缩放级别

广角 0 ~ 中景 5 ~ 特写 10

典型场景
已有角色正面三视图，想要 45° 侧脸 → 上传图 + 选右前 45° → 生成提示词 → 直接出图。
第四部分

无限画布 · 节点式自由创作

画布是 PhanthyMovie 在视频制作环节重点打磨的节点式自由创作工作区。它把"文本 / 图片 / 视频 / 音频 / 提示词 / 资产引用"统一抽象成可拖拽、可连线、可双击精修的节点，让你在一张无限大的画布上自由组织内容、串联流程，跳出四步流水线的固定路径，从一段剧本一路推进到分镜、图片、关键帧再到成片。

要点
拍摄台是按镜头精修的"工序面板"；视频工坊是任务队列驱动的"提交中心"；画布则是项目级的"自由创作沙盒"，适合把整个创作链路视觉化地串起来。

入口与新建

「无限画布」有两个入口：在「创作中心」点"无限画布"卡片，或在「项目管理」打开项目后选"无限画布"模式。点进后看到画布管理页；没有画布就点右上角"新建画布"，输入名称即可创建。

节点添加与右键菜单

新建后进入空白画布。左侧工具栏提供节点添加、素材库、历史记录、指针等；在空白处右键弹出快捷菜单：上传、添加节点、撤销、重做、粘贴。节点分四类，都支持从本地上传：

文本节点 · 剧本/提示词图片节点 · 图像素材视频节点 · 视频文件音频节点 · 音频文件

点左侧"资产库"入口，可把当前项目已挂载的资产（角色/服装/场景/关键帧等）直接拖入画布，不必重复上传。

建议
画布上曾经生成过的内容都有记录，不小心丢失或误删可以从历史里找回来，不必担心一夜白干。

完整创作演示

用一个做好的画布演示完整能力：文本节点、角色/服装/场景等图片节点、关键帧、视频，都以分组形式在一张画上串联起来，节点之间用连线表达上下游关系。

端到端示例：从剧本到成片

① 文本节点 → 图片节点：在文本节点写好角色设定（角色名、年龄、性别、身高、学校等），连线到图片节点即以这段文本为参考生图。下图角色"小葵"生成了形象和三视图（正面/四分之三/侧面）。

② 图片节点的丰富功能：选中图片节点，顶部出现一整排操作——反推、裁剪、扩图、抠图、打光、擦除、重绘、九宫格、标注、旋转镜像、下载、预览。可勾选"主图参与生成"配合提示词继续迭代。

③ 场景生图（中英文提示词都支持）：文本节点写好提示词连线到图片节点即可。下图教室场景就是用英文提示词生成的一组候选。

④ 图片节点 → 视频节点：图片/关键帧满意后，把图片节点连线到视频节点，按参考生成视频。完成后可在节点直接预览，或点开全屏播放器（支持下载、倍速、全屏）。

视频合成：多段合成为一整段成片

视频与视频之间还能做"视频合成"：把多段视频和音频连线到同一个"视频合成"节点，点"打开视频合成"进入合成界面。

合成界面是一个带时间轴的剪辑器：可对每个片段做分割、向左/向右裁剪，调整音量、变速（0.5x~4x）、时长，还支持"画中画"叠放拖拽。剪好后点右上角"导出"渲染成一整段成片。

要点
文本、图片、视频、音频节点用的都是同样的"填提示词 → 生成"逻辑，上手一处即通用。

节点打组（分组管理）

节点越堆越多时，用「打组」把相关节点归到一起统一管理。鼠标框选一批节点即打成一组，组上方浮出一排操作：

打组操作
  • 背景色：改整个组底色，用颜色区分不同角色/场景/分镜。
  • 排列：组内节点自动对齐，支持水平/垂直排列。
  • 整组执行：一键重新生成组内所有可生成节点，适合改完上游后批量刷新。
  • 解组 / 批量下载：解散分组；把组内文件一次性打包下载。
建议
画布节点一多就容易失控，养成"一个角色 / 一场戏一组，配好背景色"的习惯，后期查找、整组重生、批量导出都会省事很多。

使用建议

画布使用建议
  • 画布特别适合单段剧本 / 单镜头 demo / 临时创意验证，避开四步流程的"全量"开销。
  • 节点之间的连线关系决定上下文传递——上游内容会作为下游输入，连线时要清楚谁是输入、谁是输出。
  • 不满意时优先"追加/修改提示词"迭代，再不行才换风格/换模型，最后才整段重做，能省一大块积分。
  • 设好主图后，下游图片/关键帧/视频节点会自动沿用主图风格，显著提升一致性。
第五部分

团队子账号 · 账号与星钻

本模块主要讲账号怎么管、星钻怎么分、消耗怎么查。它分成两块——团队管理（主账号如何创建成员、分配星钻、监控消耗）和个人中心（当前账号的会员、星钻、消耗明细与设置）。一旦开始多人协作或需要对账，就是最常用的地方。

团队管理 · 主账号的后勤中枢

一个企业/工作室通常只买一个主账号充值星钻，再由主账号批量开成员子账号，把星钻按需分配给每个成员。它解决三件事：给谁开账号、给谁分多少星钻、每个人花了多少。点左侧导航栏底部"团队管理"进入。

页面顶部一排数字是团队"仪表盘"：团队项目 / 团队席位（已用·可扩容）/ 团队消耗星钻 / 团队剩余星钻 / 剩余可分配星钻（可加值），一眼看清资源用到哪了。

要点
顶部提示"团队资产库为企业版会员权益"——跨成员共享的团队资产库属于企业版能力；个人/进阶版可以创建成员、分配星钻，但团队级资产库需点"开通会员"升级后才启用。
创建成员账号
  • 在成员管理区右上角点"+ 创建成员账号"。
  • 填成员名称、备注等，系统自动派生 主账号-序号@phantymovie.com 账号。
  • 创建成功后可给他分配项目、下发星钻。
  • 把账号和初始密码交给成员即可登录（登录方式见"登录"章节）。
建议
建议在备注里写清这个账号是谁、做什么用（如"做视频"、"创作室"），人一多全靠备注区分。暂时不用的账号切到禁用，既保留数据又不占登录，比删除更稳妥。

星钻下发与回收

星钻在"主账号"和"成员账号"之间可以双向流动：主账号把星钻下发给成员消耗，成员没用完的也能被回收到主账号重新分配。在某成员行点"更多 → 调整星钻"，弹窗顶部有"下发"和"回收"两个 Tab。

注意
下发和回收都会即时改变双方余额，回收金额不能超过成员当前余额，下发金额不能超过主账号可分配额度。填数字前先看清弹窗里的两个参考值。

个人中心 · 我自己的设置与账单

个人中心是当前登录账号自己的"设置与账单"面板，主账号和成员账号都有。点界面右上角头像展开下拉面板，顶部是账号头像+手机号（脱敏）、会员版本与剩余天数（可升级/续费）、星钻余额（醒目大字）。

常用入口包括：账号设置、交易记录、我的消耗明细、AI 水印设置、退出登录。主账号还会看到"您的团队成员账户中还有 X 星钻未使用" + 一键"转回主账号"。

我的消耗明细 · 对账工具

这是排查"星钻花在哪了"的对账工具，按任务逐条列出明细。顶部时间筛选（今天/昨天/本周/本月…）+ "全部类型/全部状态"下拉；表格字段有功能点、标题、模型、项目、状态、任务时间、耗时、星钻。

建议
对账小技巧：怀疑某天星钻掉得快，把时间筛成"昨天"，按"星钻"列从高到低扫一眼，最贵的那几笔（通常是"生视频"）一目了然。失败但也扣了星钻的任务，可据此截图找管理员核对。
